# py-video-thumbnail

Create thumbnail from a video (default 4x4).

## Environment
```bash
$ virtualenv -p python3.6 venv
$ source venv/bin/activate
$ pip install opencv-python
```

## Usage

```bash
$ python pvt.py BigBuckBunny.mp4
# Or
$ python pvt.py *.mp4
```
